# このリポジトリについて(AI向け目次)

このファイル自体には作業内容の詳細は書かない。以下の振り分けに従って、目的のディレクトリへ直接進むこと。

## どこを見ればいいか

- **特定プロジェクト(todo-app など)で作業している** → `projects/<project名>/` を見る。無ければそのプロジェクトのリポジトリ(`/opt/ops/apps/<project名>/`)の `CLAUDE.md` / `planning/` を直接見る。
- **homenas サーバ自体(ハード・ネットワーク・ストレージ・バックアップ・各コンテナの運用)について知りたい** → `homenas/README.md` と `homenas/CLAUDE.md` を見る。
- **プロジェクトを問わず、obukata と AI エージェントの会話・作業で常に踏まえるべきこと** → `ai-agent-policy.md` を見る。呼び出されたディレクトリに関わらず、必ず一度は目を通すこと。

## このリポジトリの編集ルール

- `docs/`(このリポジトリ)は obukata が管理する。`aiagent` は実体との乖離が分かった場合に更新してよいが、(1) 変更は必ず commit し理由を書く、(2) `/opt/ops/AGENTS.md` 自体は対象外、(3) 秘密情報は書かない。
- push は topic ブランチのみ。`main` への直 push はしない(PR を作成する)。
